# Much Monitor Pro (Swift Native Version)

This is the native macOS version of Much Monitor Pro built with SwiftUI and AVFoundation.

## Features
- **Native Performance**: Extremely low CPU/RAM usage.
- **AVFoundation**: Superior iPhone/Continuity Camera detection.
- **SwiftUI**: Modern, smooth macOS-native UI with hardware acceleration.

## Requirements
- macOS 14.0 or later
- Xcode 15 or later

## How to Run
1. Open terminal in this directory.
2. Run `swift run` or open the folder in Xcode.
3. To build a standalone `.app`, use Xcode: **Product > Archive**.
