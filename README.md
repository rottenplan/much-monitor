# Much Monitor Pro

A professional monitor calibration suite with dual implementations.

## Project Structure

### [much-monitor-python/](file:///Users/muchdas/Documents/Anti%20Gravity/much-monitor/much-monitor-python)
The original implementation using Python, Tkinter, and OpenCV.
- **Main App**: `main_gui.py`
- **Logic**: `calibration_logic.py`
- **Requirements**: Python 3.10+, OpenCV, Pillow.

### [much-monitor-swift/](file:///Users/muchdas/Documents/Anti%20Gravity/much-monitor/much-monitor-swift)
The new native macOS implementation using SwiftUI and AVFoundation.
- **Frameworks**: SwiftUI, AVFoundation, Vision.
- **Status**: Core UI and Camera Engine ported.
- **Requirements**: macOS 14+, Xcode 15+.
